# EX-04-2b
## AIM:
Write a C program to check whether profit or loss using simple if statement
## ALGORITHM:
1. Include Header Files
2. Declare Variables
3. Get Input
4. Calculate Profit or Loss
5. End the program
## PROGRAM:
```
#include<stdio.h>
int main(){
int a,b;
scanf("%d%d",&a,&b);
if(a<b){
printf("Profit!!");
}
if(b<a)
{
printf("Loss!!");
}
return 0;
}
```
